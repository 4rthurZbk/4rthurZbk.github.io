ZIRK programming language official documentation // Data types

1. Numbers


Numbers are used to count, bruh. To assin a number variable use _yan_ keyword.

Program exemple:

_yan_ a = 11

a + 1

owo a

Program output:

12

2. Letters


Letters are used to do stuff, lmao. To assin a letter variable use _shyan_ keyword (like yan but with s (s like string) idk).

_shyan_ a = Hello_world

a + !
